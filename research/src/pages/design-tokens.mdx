# Design Tokens

A foundational aspect to user-interfaces is the visual design of experiences. This section aims to describe common design token names, uses, and values as they are used in visual design of user-interfaces, in a manner that can be used and shared across products and platforms.

## Goals

- to support accessible enterprise web applications
- capture general design tokens for most common UI
- cross-platform usage

## Non-goals

- enumerate an exhaustive set of design tokens
- describe nuances of marketing, landing, and other highly unique webpages
- define subjective style attributes / names

### Stage 3

| Name        | Description                                                                                                                    | Type   |
| :---------- | :----------------------------------------------------------------------------------------------------------------------------- | :----- |
| density     | Describes relative spacial and dimensional aspects of UI. A lower number denotes a more compact UI.                            | number |
| design-unit | Describes the foundational sizing unit for a UI. Spacial and dimensional properties will generally be a multiple of this unit. | number |

### Stage 2
